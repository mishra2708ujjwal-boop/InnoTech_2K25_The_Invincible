# InnoTech_2K25_The_Invincible
InnoTech 2025 CSE department level hackathon of KIET College

The Club Universe of KIET – "The Invincible"

A campus-wide portal for discovering, joining, and engaging with all student clubs, domains, and events at KIET.

Overview:-

This web application provides an interactive, modern interface for students to explore technical, cultural, sports, social service, and entrepreneurship clubs at KIET. Core features include advanced club filtering, detailed club cards, animated UI, event calendar, announcements, student ratings, reviews, and login system – all in a rich, responsive design.

Features:- 

Beautiful UI – Modern dark/light themes with premium gradients and smooth animations.

Interactive Navigation Bar – Access login, theme switcher, info, queries, events, and calendar.

Club Filtering – Browse clubs by domain and subdomain.

Search – Instantly search clubs by name/description.

Event Calendar – Monthly calendar highlighting club events.

Club Profiles – Modal with club logo, description, contact info, president/coordinator details, Instagram link, rating, reviews, activities, achievements, registration dates, and join button.

Student Reviews & Ratings – Submit and view detailed reviews for each club.

Announcements Section – Sticky, draggable notifications for club events or updates.

Login Modal – Simulate member login via name, university email, roll number.

Queries Modal – Submit queries by category via a dedicated form.

Fully responsive – Works great on desktop and mobile.
